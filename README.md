## Rock RMS Parallax Photo Wall

This uses flex box to create a photo wall from images stored in a content channel. It sets the photo wall up as a background with parallax effect. The content is centered on top of the photo wall background and scrolls with the page.


## To use:

1. Add the HTML to your Content Channel View Content Component
2. Add the CSS to Advanced Settings.
3. Update the html with your theme specific styles like background colors, buttons, etc.
